## 📂 Projets principaux

### 1. **🎴 Inscryption** - Jeu de cartes stratégique

**Projet encadré - Java | Binôme | 5 semaines**

Un jeu de cartes stratégique imitant le jeu *Inscryption*, où vous affrontez un adversaire dans une partie tactique avec gestion d'économie de ressources.

**Caractéristiques principales :**
- Système de jeu complet avec plateau 2×4 et gestion du score
- 10 types de cartes animales avec mécaniques variées (volant, sacrifices, os)
- 2 obstacles destructibles (rocher, sapin)
- IA adversaire déterministe (non-aléatoire pour faciliter le test)
- Mécaniques complexes : coûts en sang (cartes sacrifiées) et en os (cartes mortes)
- Mode progression : 3 parties gagnées pour remporter le jeu
- Sélection de nouvelles cartes à chaque fin de partie

**Architecture & Points forts :**
- Architecture orientée objet rigoureuse avec diagrammes UML mis à jour chaque semaine
- Séparation claire des responsabilités (Modèle/Vue)
- Suite de tests complète (JUnit) : attaques, score, placement, pioches, victoires
- Gestion robuste des erreurs et saisies utilisateur invalides
- Code commenté respectant les P21 Guidelines
- Rendus hebdomadaires sur Git (branches `rendu1-5`)

**Technologies** : Java, JUnit, Git

**État** : ✅ Complet et testé

### 2. **🚀 Project Stargate** - Gestion de missions spatiales

**Projet système d'information - C# WinForms | Binôme | Durée variable**

Application complète de gestion de missions d'exploration spatiale pour la recherche de minerai stratégique (DataBaz) et la diplomatie avec espèces extraterrestres.

**7 volets fonctionnels :**

| # | Volet | Description |
|---|-------|-------------|
| **1** | 📊 Tableau de bord | Vue synthétique des missions (en cours, passées, à venir) avec génération de PDF de bilan |
| **2** | ➕ Création mission | Interface d'authentification admin + création de missions avec feuille de route |
| **3** | 📋 Détails mission | Récapitulatif complet : équipage, objectifs, budget, captures, événements, contacts |
| **4** | 📖 Journal de bord | Visualisation timeline des événements (liaison de données, sans accès BD) |
| **5** | 👽 Races répertoriées | Liste des espèces (alliées/ennemies) avec filtres et caractéristiques détaillées |
| **6** | 🪐 Informations planètes | Données planétaires : atmosphère, gravité, espèces présentes, missions effectuées |
| **7** | 📈 Statistiques | 5 requêtes SQL avancées : personnes, budgets, missions par planète, dépenses, informateurs |

**Architecture & Points forts :**
- Base SQLite robuste avec schéma relationnel complexe (17 tables)
- Gestion de connexion centralisée (`Connexion.cs`)
- DataSet indépendant (`mesDatas.cs`) pour partage d'état
- Authentification avec hachage de mot de passe
- Transactions SQL pour l'intégrité des données
- Requêtes SQL avancées (agrégations, jointures, sous-requêtes)
- Génération de rapports PDF dynamiques
- Interface ergonomique et professionnelle
- Code commenté et lisible

**Technologies** : C#, WinForms, SQLite, SQL, PDF generation

**État** : 🔄 En développement (volets 1-7)
